  ## Getting Started
  
  ### Installation
  
  1. Fork the Repo from the [Repository](https://github.com//)
2. Clone the repo
   ```sh
   git clone git@github.com:/.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run npm start
   ```sh
   npm start
   ```
